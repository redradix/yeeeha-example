⌨️ Interacción del usuario con git

Una vez que el usuario comience a utilizar git, se enviará a un servidor proxy los siguientes datos:

  • userName --> Usuario de git
  • event_name --> Push y pull
  • proyectName --> Nombre del repositorio
  • day --> Día
  • hour --> Hora
